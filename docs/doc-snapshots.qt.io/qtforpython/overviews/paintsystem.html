Paint System¶

A system for painting on the screen or on print devices using the same API

A system for painting on the screen or on print devices using the same API

Qt’s paint system enables painting on screen and print devices using the same API, and is primarily based on the PySide2.QtGui.QPainter , PySide2.QtGui.QPaintDevice , and PySide2.QtGui.QPaintEngine classes.

PySide2.QtGui.QPainter is used to perform drawing operations, PySide2.QtGui.QPaintDevice is an abstraction of a two-dimensional space that can be painted on using a PySide2.QtGui.QPainter , and PySide2.QtGui.QPaintEngine provides the interface that the painter uses to draw onto different types of devices. The PySide2.QtGui.QPaintEngine class is used internally by PySide2.QtGui.QPainter and PySide2.QtGui.QPaintDevice , and is hidden from application programmers unless they create their own device type.

The main benefit of this approach is that all painting follows the same painting pipeline making it easy to add support for new features and providing default implementations for unsupported ones.
