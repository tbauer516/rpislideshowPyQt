Trivial Wizard Example¶

Trivial Wizard example illustrates how to create a linear three-page registration wizard using three instances of QWizardPage and one instance of QWizard

The Trivial Wizard example illustrates how to create a linear three-page registration wizard using three instances of PySide2.QtWidgets.QWizardPage and one instance of PySide2.QtWidgets.QWizard .

Introduction Page¶

The introduction page is created with the createIntroPage() function where a PySide2.QtWidgets.QWizardPage is created and its title is set to “Introduction”. A PySide2.QtWidgets.QLabel is used to hold the description of page . A PySide2.QtWidgets.QVBoxLayout is used to hold the label . This page is returned when the createIntroPage() function is called.

def createIntroPage(self):
    page = QWizardPage()
    page.setTitle("Introduction")

    label = QLabel("This wizard will help you register your copy of Super Product Two.")
    label.setWordWrap(True)

    layout = QVBoxLayout()
    layout.addWidget(label)
    page.setLayout(layout)

    return page

Registration Page¶

The registration page is created with the createRegistrationPage() function. PySide2.QtWidgets.QLineEdit objects are used to allow the user to input a name and an e-mail address. A PySide2.QtWidgets.QGridLayout is used to hold the PySide2.QtWidgets.QLabel and PySide2.QtWidgets.QLineEdit objects.

QWizardPage *createRegistrationPage()



    QWizardPage *page = new QWizardPage;
    page->setTitle("Registration");
    page->setSubTitle("Please fill both fields.");

    QLabel *nameLabel = new QLabel("Name:");
    QLineEdit *nameLineEdit = new QLineEdit;

    QLabel *emailLabel = new QLabel("Email address:");
    QLineEdit *emailLineEdit = new QLineEdit;

    QGridLayout *layout = new QGridLayout;
    layout->addWidget(nameLabel, 0, 0);
    layout->addWidget(nameLineEdit, 0, 1);
    layout->addWidget(emailLabel, 1, 0);
    layout->addWidget(emailLineEdit, 1, 1);
    page->setLayout(layout);

    return page;

Conclusion Page¶

The conclusion page is created in the createConclusionPage() function. This function’s content is similar to createIntroPage() . A PySide2.QtWidgets.QLabel is used to inform the user that the registration process has completed successfully.

def createConclusionPage(self):



    QWizardPage *page = new QWizardPage;
    page->setTitle("Conclusion");

    QLabel *label = new QLabel("You are now successfully registered. Have a "
                               "nice day!");
    label->setWordWrap(true);

    QVBoxLayout *layout = new QVBoxLayout;
    layout->addWidget(label);
    page->setLayout(layout);

    return page;

main()

Function¶

The main() function instantiates a PySide2.QtWidgets.QWizard object, wizard , and adds all three PySide2.QtWidgets.QWizardPage objects to it. The wizard window title is set to “Trivial Wizard” and its show() function is invoked to display it.

def main():
    app = QApplication(sys.argv)

    translatorFileName = "qt_"
    translatorFileName += QLocale.system().name()
    translator = QTranslator(app)
    if translator.load(translatorFileName, QLibraryInfo.location(QLibraryInfo.TranslationsPath)):
        app.installTranslator(translator)

    wizard = QWizard()
    wizard.addPage(createIntroPage())
    wizard.addPage(createRegistrationPage())
    wizard.addPage(createConclusionPage())

    wizard.setWindowTitle("Trivial Wizard")
    wizard.show()

    return app.exec_()

if __name__ == "__main__":
    main()
