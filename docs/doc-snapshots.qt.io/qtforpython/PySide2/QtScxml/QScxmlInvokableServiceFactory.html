Detailed Description¶

The PySide2.QtScxml.QScxmlInvokableServiceFactory class creates invokable service instances.

Each service instance represents an <invoke> element in the SCXML document. Each time the service is actually invoked, a new instance of PySide2.QtScxml.QScxmlInvokableService is created.
