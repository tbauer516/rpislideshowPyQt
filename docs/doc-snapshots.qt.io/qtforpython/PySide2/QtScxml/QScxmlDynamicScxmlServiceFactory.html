QScxmlDynamicScxmlServiceFactory¶

Detailed Description¶

The PySide2.QtScxml.QScxmlDynamicScxmlServiceFactory class creates SCXML service instances from documents loaded at runtime.

Dynamically resolved services are used when loading SCXML content from files that a parent state machine requests at runtime, via the srcexpr attribute in the <invoke> element.

class PySide2.QtScxml.QScxmlDynamicScxmlServiceFactory(invokeInfo, names, parameters[, parent=nullptr])¶
Parameters:
  • invokeInfo – PySide2.QtScxml.QScxmlExecutableContent::InvokeInfo
  • names –
  • parameters –
  • parent – PySide2.QtCore.QObject

Creates a factory for dynamically resolved services, passing the attributes of the <invoke> element as invokeInfo , any <param> child elements as parameters , the content of the names attribute as names , and the PySide2.QtCore.QObject parent parent .
