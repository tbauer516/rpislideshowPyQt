Detailed Description¶

The PySide2.QtHelp.QHelpEngine class provides access to contents and indices of the help engine.
class PySide2.QtHelp.QHelpEngine(collectionFile[, parent=nullptr])¶
Parameters:
  • collectionFile – unicode
  • parent – PySide2.QtCore.QObject

Constructs a new help engine with the given parent . The help engine uses the information stored in the collectionFile for providing help. If the collection file does not already exist, it will be created.
