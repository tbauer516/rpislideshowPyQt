QCameraViewfinder¶

Detailed Description¶

The PySide2.QtMultimediaWidgets.QCameraViewfinder class provides a camera viewfinder widget.

camera = new QCamera;

viewfinder = new QCameraViewfinder();
viewfinder->show();

camera->setViewfinder(viewfinder);

imageCapture = new QCameraImageCapture(camera);

camera->setCaptureMode(QCamera::CaptureStillImage);
camera->start();
class PySide2.QtMultimediaWidgets.QCameraViewfinder([parent=nullptr])¶
Parameters:parent – PySide2.QtWidgets.QWidget

Constructs a new camera viewfinder widget.

The parent is passed to QVideoWidget .
