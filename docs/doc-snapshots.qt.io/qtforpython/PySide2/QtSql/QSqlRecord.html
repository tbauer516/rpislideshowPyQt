Detailed Description¶

The PySide2.QtSql.QSqlRecord class encapsulates a database record.

The PySide2.QtSql.QSqlRecord class encapsulates the functionality and characteristics of a database record (usually a row in a table or view within the database). PySide2.QtSql.QSqlRecord supports adding and removing fields as well as setting and retrieving field values.

The values of a record’s fields can be set by name or position with PySide2.QtSql.QSqlRecord.setValue() ; if you want to set a field to null use PySide2.QtSql.QSqlRecord.setNull() . To find the position of a field by name use PySide2.QtSql.QSqlRecord.indexOf() , and to find the name of a field at a particular position use PySide2.QtSql.QSqlRecord.fieldName() . Use PySide2.QtSql.QSqlRecord.field() to retrieve a PySide2.QtSql.QSqlField object for a given field. Use PySide2.QtSql.QSqlRecord.contains() to see if the record contains a particular field name.

When queries are generated to be executed on the database only those fields for which PySide2.QtSql.QSqlRecord.isGenerated() is true are included in the generated SQL.

A record can have fields added with PySide2.QtSql.QSqlRecord.append() or PySide2.QtSql.QSqlRecord.insert() , replaced with PySide2.QtSql.QSqlRecord.replace() , and removed with PySide2.QtSql.QSqlRecord.remove() . All the fields can be removed with PySide2.QtSql.QSqlRecord.clear() . The number of fields is given by PySide2.QtSql.QSqlRecord.count() ; all their values can be cleared (to null) using PySide2.QtSql.QSqlRecord.clearValues() .
