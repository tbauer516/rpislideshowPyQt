Detailed Description¶

The PySide2.QtQuick.QSGBasicGeometryNode class serves as a baseclass for geometry based nodes.

The PySide2.QtQuick.QSGBasicGeometryNode class should not be used by itself. It is only encapsulates shared functionality between the PySide2.QtQuick.QSGGeometryNode and PySide2.QtQuick.QSGClipNode classes.

Note

All classes with QSG prefix should be used solely on the scene graph’s rendering thread. See Scene Graph and Rendering for more information.

PySide2.QtQuick.QSGBasicGeometryNode.setGeometry(geometry)¶
Parameters:geometry – PySide2.QtQuick.QSGGeometry

Sets the geometry of this node to geometry .

If the node has the flag QSGNode.OwnsGeometry set, it will also delete the geometry object it is pointing to. This flag is not set by default.

If the geometry is changed without calling again, the user must also mark the geometry as dirty using QSGNode.markDirty() .
