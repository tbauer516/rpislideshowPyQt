QSGDynamicTexture¶

Detailed Description¶

The PySide2.QtQuick.QSGDynamicTexture class serves as a baseclass for dynamically changing textures, such as content that is rendered to FBO’s.

To update the content of the texture, call PySide2.QtQuick.QSGDynamicTexture.updateTexture() explicitly. Simply calling PySide2.QtQuick.QSGTexture.bind() will not update the texture.

Note

All classes with QSG prefix should be used solely on the scene graph’s rendering thread. See Scene Graph and Rendering for more information.

class PySide2.QtQuick.QSGDynamicTexture¶
PySide2.QtQuick.QSGDynamicTexture.updateTexture()¶
Return type:PySide2.QtCore.bool

Call this function to explicitly update the dynamic texture. Calling PySide2.QtQuick.QSGTexture.bind() will bind the content that was previously updated.

The function returns true if the texture was changed as a resul of the update; otherwise returns false.
