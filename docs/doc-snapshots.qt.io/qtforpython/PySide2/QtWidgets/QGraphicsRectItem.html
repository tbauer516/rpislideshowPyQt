Detailed Description¶

The PySide2.QtWidgets.QGraphicsRectItem class provides a rectangle item that you can add to a PySide2.QtWidgets.QGraphicsScene .

To set the item’s rectangle, pass a PySide2.QtCore.QRectF to PySide2.QtWidgets.QGraphicsRectItem ‘s constructor, or call the PySide2.QtWidgets.QGraphicsRectItem.setRect() function. The PySide2.QtWidgets.QGraphicsRectItem.rect() function returns the current rectangle.

PySide2.QtWidgets.QGraphicsRectItem uses the rectangle and the pen width to provide a reasonable implementation of PySide2.QtWidgets.QGraphicsRectItem.boundingRect() , PySide2.QtWidgets.QGraphicsRectItem.shape() , and PySide2.QtWidgets.QGraphicsRectItem.contains() . The PySide2.QtWidgets.QGraphicsRectItem.paint() function draws the rectangle using the item’s associated pen and brush, which you can set by calling the PySide2.QtWidgets.QAbstractGraphicsShapeItem.setPen() and PySide2.QtWidgets.QAbstractGraphicsShapeItem.setBrush() functions.

Note

The rendering of invalid rectangles, such as those with negative widths or heights, is undefined. If you cannot be sure that you are using valid rectangles (for example, if you are creating rectangles using data from an unreliable source) then you should use QRectF.normalized() to create normalized rectangles, and use those instead.
