Detailed Description¶

The PySide2.QtWidgets.QGraphicsAnchor class represents an anchor between two items in a PySide2.QtWidgets.QGraphicsAnchorLayout .

The graphics anchor provides an API that enables you to query and manipulate the properties an anchor has. When an anchor is added to the layout with QGraphicsAnchorLayout.addAnchor() , a PySide2.QtWidgets.QGraphicsAnchor instance is returned where the properties are initialized to their default values. The properties can then be further changed, and they will be picked up the next time the layout is activated.
