Detailed Description¶

The PySide2.QtWidgets.QGraphicsLineItem class provides a line item that you can add to a PySide2.QtWidgets.QGraphicsScene .

To set the item’s line, pass a PySide2.QtCore.QLineF to PySide2.QtWidgets.QGraphicsLineItem ‘s constructor, or call the PySide2.QtWidgets.QGraphicsLineItem.setLine() function. The PySide2.QtWidgets.QGraphicsLineItem.line() function returns the current line. By default the line is black with a width of 0, but you can change this by calling PySide2.QtWidgets.QGraphicsLineItem.setPen() .

PySide2.QtWidgets.QGraphicsLineItem uses the line and the pen width to provide a reasonable implementation of PySide2.QtWidgets.QGraphicsLineItem.boundingRect() , PySide2.QtWidgets.QGraphicsLineItem.shape() , and PySide2.QtWidgets.QGraphicsLineItem.contains() . The PySide2.QtWidgets.QGraphicsLineItem.paint() function draws the line using the item’s associated pen.
