Detailed Description¶

The PySide2.QtWidgets.QAccessibleWidget class implements the PySide2.QtGui.QAccessibleInterface for QWidgets.

This class is part of Accessibility for QWidget Applications.

This class is convenient to use as a base class for custom implementations of QAccessibleInterfaces that provide information about widget objects.

The class provides functions to retrieve the PySide2.QtWidgets.QAccessibleWidget.parentObject() (the widget’s parent widget), and the associated PySide2.QtWidgets.QAccessibleWidget.widget() . Controlling signals can be added with PySide2.QtWidgets.QAccessibleWidget.addControllingSignal() , and setters are provided for various aspects of the interface implementation, for example setValue(), setDescription(), setAccelerator(), and setHelp().
