Detailed Description¶

The PySide2.QtWidgets.QGraphicsSceneHoverEvent class provides hover events in the graphics view framework.

When a PySide2.QtWidgets.QGraphicsView receives a PySide2.QtGui.QHoverEvent event, it translates it into PySide2.QtWidgets.QGraphicsSceneHoverEvent . The event is then forwarded to the PySide2.QtWidgets.QGraphicsScene associated with the view.
