Detailed Description¶

The PySide2.QtWidgets.QAbstractGraphicsShapeItem class provides a common base for all path items.

This class does not fully implement an item by itself; in particular, it does not implement PySide2.QtWidgets.QGraphicsItem.boundingRect() and PySide2.QtWidgets.QGraphicsItem.paint() , which are inherited by PySide2.QtWidgets.QGraphicsItem .

You can subclass this item to provide a simple base implementation of accessors for the item’s pen and brush.
