Detailed Description¶

The PySide2.QtWidgets.QFileIconProvider class provides file icons for the PySide2.QtWidgets.QDirModel and the PySide2.QtWidgets.QFileSystemModel classes.
class PySide2.QtWidgets.QFileIconProvider¶

Constructs a file icon provider.

PySide2.QtWidgets.QFileIconProvider.Option¶
Constant Description
QFileIconProvider.DontUseCustomDirectoryIcons Always use the default directory icon. Some platforms allow the user to set a different icon. Custom icon lookup cause a big performance impact over network or removable drives.
