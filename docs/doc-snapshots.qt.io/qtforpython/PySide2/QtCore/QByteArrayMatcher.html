Detailed Description¶

The PySide2.QtCore.QByteArrayMatcher class holds a sequence of bytes that can be quickly matched in a byte array.

This class is useful when you have a sequence of bytes that you want to repeatedly match against some byte arrays (perhaps in a loop), or when you want to search for the same sequence of bytes multiple times in the same byte array. Using a matcher object and PySide2.QtCore.QByteArrayMatcher.indexIn() is faster than matching a plain PySide2.QtCore.QByteArray with QByteArray.indexOf() if repeated matching takes place. This class offers no benefit if you are doing one-off byte array matches.

Create the PySide2.QtCore.QByteArrayMatcher with the PySide2.QtCore.QByteArray you want to search for. Then call PySide2.QtCore.QByteArrayMatcher.indexIn() on the PySide2.QtCore.QByteArray that you want to search.

  • len – PySide2.QtCore.int

Return type:

PySide2.QtCore.int

Searches the char string str , which has length len , from byte position from (default 0, i.e. from the first byte), for the byte array PySide2.QtCore.QByteArrayMatcher.pattern() that was set in the constructor or in the most recent call to PySide2.QtCore.QByteArrayMatcher.setPattern() . Returns the position where the PySide2.QtCore.QByteArrayMatcher.pattern() matched in str , or -1 if no match was found.
