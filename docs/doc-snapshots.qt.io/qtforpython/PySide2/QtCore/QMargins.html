Detailed Description¶

The PySide2.QtCore.QMargins class defines the four margins of a rectangle.

QMargin defines a set of four margins; left, top, right and bottom, that describe the size of the borders surrounding a rectangle.

The PySide2.QtCore.QMargins.isNull() function returns true only if all margins are set to zero.

QMargin objects can be streamed as well as compared.
