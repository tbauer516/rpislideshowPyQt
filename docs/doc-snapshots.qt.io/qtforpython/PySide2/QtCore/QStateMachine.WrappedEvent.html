QStateMachine.WrappedEvent¶

Note

This class was introduced in Qt 4.6.

Synopsis¶

Functions¶

  • def event ()
  • def object ()

Detailed Description¶

The WrappedEvent class inherits PySide2.QtCore.QEvent and holds a clone of an event associated with a PySide2.QtCore.QObject .

A wrapped event is generated by a PySide2.QtCore.QStateMachine in response to a Qt event. The PySide2.QtCore.QEventTransition class provides a transition associated with a such an event. QStateMachine.WrappedEvent is part of The State Machine Framework .

The PySide2.QtCore.QStateMachine::WrappedEvent.object() function returns the object that generated the event. The PySide2.QtCore.QStateMachine::WrappedEvent.event() function returns a clone of the original event.
