Detailed Description¶

The PySide2.QtCore.QBasicTimer class provides timer events for objects.

This is a fast, lightweight, and low-level class used by Qt internally. We recommend using the higher-level PySide2.QtCore.QTimer class rather than this class if you want to use timers in your applications. Note that this timer is a repeating timer that will send subsequent timer events unless the PySide2.QtCore.QBasicTimer.stop() function is called.

To use this class, create a PySide2.QtCore.QBasicTimer , and call its PySide2.QtCore.QBasicTimer.start() function with a timeout interval and with a pointer to a PySide2.QtCore.QObject subclass. When the timer times out it will send a timer event to the PySide2.QtCore.QObject subclass. The timer can be stopped at any time using PySide2.QtCore.QBasicTimer.stop() . PySide2.QtCore.QBasicTimer.isActive() returns true for a timer that is running; i.e. it has been started, has not reached the timeout time, and has not been stopped. The timer’s ID can be retrieved using PySide2.QtCore.QBasicTimer.timerId() .

The Wiggly example uses PySide2.QtCore.QBasicTimer to repaint a widget at regular intervals.
