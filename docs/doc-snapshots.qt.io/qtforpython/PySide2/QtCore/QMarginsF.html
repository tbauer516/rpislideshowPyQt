Detailed Description¶

The PySide2.QtCore.QMarginsF class defines the four margins of a rectangle.

PySide2.QtCore.QMarginsF defines a set of four margins; left, top, right and bottom, that describe the size of the borders surrounding a rectangle.

The PySide2.QtCore.QMarginsF.isNull() function returns true only if all margins are set to zero.

PySide2.QtCore.QMarginsF objects can be streamed as well as compared.
