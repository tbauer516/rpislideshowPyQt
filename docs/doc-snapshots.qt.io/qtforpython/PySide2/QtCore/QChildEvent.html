Detailed Description¶

The PySide2.QtCore.QChildEvent class contains event parameters for child object events.

Child events are sent immediately to objects when children are added or removed.

In both cases you can only rely on the child being a PySide2.QtCore.QObject (or, if QObject.isWidgetType() returns true , a PySide2.QtWidgets.QWidget ). This is because in the QEvent.ChildAdded case the child is not yet fully constructed; in the QEvent.ChildRemoved case it might have already been destructed.

The handler for these events is QObject.childEvent() .

class PySide2.QtCore.QChildEvent(type, child)¶
Parameters:
  • type – PySide2.QtCore.QEvent.Type
  • child – PySide2.QtCore.QObject

Constructs a child event object of a particular type for the child .

type can be QEvent.ChildAdded , QEvent.ChildRemoved , or QEvent.ChildPolished .
