Detailed Description¶

The PySide2.QtGui.QTextItem class provides all the information required to draw text in a custom paint engine.

When you reimplement your own paint engine, you must reimplement QPaintEngine.drawTextItem() , a function that takes a PySide2.QtGui.QTextItem as one of its arguments.

class PySide2.QtGui.QTextItem¶
PySide2.QtGui.QTextItem.RenderFlag¶
Constant Description
QTextItem.RightToLeft Render the text from right to left.
QTextItem.Overline Paint a line above the text.
QTextItem.Underline Paint a line under the text.
QTextItem.StrikeOut Paint a line through the text.
