Detailed Description¶

The PySide2.QtGui.QTextBlockFormat class provides formatting information for blocks of text in a PySide2.QtGui.QTextDocument .

A document is composed of a list of blocks, represented by PySide2.QtGui.QTextBlock objects. Each block can contain an item of some kind, such as a paragraph of text, a table, a list, or an image. Every block has an associated PySide2.QtGui.QTextBlockFormat that specifies its characteristics.

To cater for left-to-right and right-to-left languages you can set a block’s direction with setDirection(). Paragraph alignment is set with PySide2.QtGui.QTextBlockFormat.setAlignment() . Margins are controlled by PySide2.QtGui.QTextBlockFormat.setTopMargin() , PySide2.QtGui.QTextBlockFormat.setBottomMargin() , PySide2.QtGui.QTextBlockFormat.setLeftMargin() , PySide2.QtGui.QTextBlockFormat.setRightMargin() . Overall indentation is set with PySide2.QtGui.QTextBlockFormat.setIndent() , the indentation of the first line with PySide2.QtGui.QTextBlockFormat.setTextIndent() .

Line spacing is set with PySide2.QtGui.QTextBlockFormat.setLineHeight() and retrieved via PySide2.QtGui.QTextBlockFormat.lineHeight() and PySide2.QtGui.QTextBlockFormat.lineHeightType() . The types of line spacing available are in the QTextBlockFormat.LineHeightTypes enum.

Line breaking can be enabled and disabled with PySide2.QtGui.QTextBlockFormat.setNonBreakableLines() .

The brush used to paint the paragraph’s background is set with PySide2.QtGui.QTextFormat.setBackground() , and other aspects of the text’s appearance can be customized by using the PySide2.QtGui.QTextFormat.setProperty() function with the OutlinePen , ForegroundBrush , and BackgroundBrush QTextFormat.Property values.

If a text block is part of a list, it can also have a list format that is accessible with the listFormat() function.

PySide2.QtGui.QTextBlockFormat.LineHeightTypes¶

This enum describes the various types of line spacing support paragraphs can have.

Constant Description
QTextBlockFormat.SingleHeight This is the default line height: single spacing.
QTextBlockFormat.ProportionalHeight This sets the spacing proportional to the line (in percentage). For example, set to 200 for double spacing.
QTextBlockFormat.FixedHeight This sets the line height to a fixed line height (in pixels).
QTextBlockFormat.MinimumHeight This sets the minimum line height (in pixels).
QTextBlockFormat.LineDistanceHeight This adds the specified height between lines (in pixels).

PySide2.QtGui.QTextBlockFormat.lineHeight(scriptLineHeight, scaling)
Parameters:
  • scriptLineHeight – PySide2.QtCore.qreal
  • scaling – PySide2.QtCore.qreal
Return type:

PySide2.QtCore.qreal

Returns the height of the lines in the paragraph based on the height of the script line given by scriptLineHeight and the specified scaling factor.

The value that is returned is also dependent on the given LineHeightType of the paragraph as well as the LineHeight setting that has been set for the paragraph.

The scaling is needed for heights that include a fixed number of pixels, to scale them appropriately for printing.

PySide2.QtGui.QTextBlockFormat.setIndent(indent)¶
Parameters:indent – PySide2.QtCore.int

Sets the paragraph’s indentation . Margins are set independently of indentation with PySide2.QtGui.QTextBlockFormat.setLeftMargin() and PySide2.QtGui.QTextBlockFormat.setTextIndent() . The indentation is an integer that is multiplied with the document-wide standard indent, resulting in the actual indent of the paragraph.
