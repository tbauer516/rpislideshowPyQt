Detailed Description¶

The PySide2.QtGui.QPdfWriter class is a class to generate PDFs that can be used as a paint device.

PySide2.QtGui.QPdfWriter generates PDF out of a series of drawing commands using PySide2.QtGui.QPainter . The PySide2.QtGui.QPdfWriter.newPage() method can be used to create several pages.
