Detailed Description¶

The PySide2.QtGui.QPaintDevice class is the base class of objects that can be painted on with PySide2.QtGui.QPainter .

A paint device is an abstraction of a two-dimensional space that can be drawn on using a PySide2.QtGui.QPainter . Its default coordinate system has its origin located at the top-left position. X increases to the right and Y increases downwards. The unit is one pixel.

The drawing capabilities of PySide2.QtGui.QPaintDevice are currently implemented by the PySide2.QtWidgets.QWidget , PySide2.QtGui.QImage , PySide2.QtGui.QPixmap , PySide2.QtOpenGL.QGLPixelBuffer , PySide2.QtGui.QPicture , and PySide2.QtPrintSupport.QPrinter subclasses.

To implement support for a new backend, you must derive from PySide2.QtGui.QPaintDevice and reimplement the virtual PySide2.QtGui.QPaintDevice.paintEngine() function to tell PySide2.QtGui.QPainter which paint engine should be used to draw on this particular device. Note that you also must create a corresponding paint engine to be able to draw on the device, i.e derive from PySide2.QtGui.QPaintEngine and reimplement its virtual functions.

Warning

Qt requires that a PySide2.QtGui.QGuiApplication object exists before any paint devices can be created. Paint devices access window system resources, and these resources are not initialized before an application object is created.

The PySide2.QtGui.QPaintDevice class provides several functions returning the various device metrics: The PySide2.QtGui.QPaintDevice.depth() function returns its bit depth (number of bit planes). The PySide2.QtGui.QPaintDevice.height() function returns its height in default coordinate system units (e.g. pixels for PySide2.QtGui.QPixmap and PySide2.QtWidgets.QWidget ) while PySide2.QtGui.QPaintDevice.heightMM() returns the height of the device in millimeters. Similiarily, the PySide2.QtGui.QPaintDevice.width() and PySide2.QtGui.QPaintDevice.widthMM() functions return the width of the device in default coordinate system units and in millimeters, respectively. Alternatively, the protected PySide2.QtGui.QPaintDevice.metric() function can be used to retrieve the metric information by specifying the desired QPaintDevice.PaintDeviceMetric as argument.

The PySide2.QtGui.QPaintDevice.logicalDpiX() and PySide2.QtGui.QPaintDevice.logicalDpiY() functions return the horizontal and vertical resolution of the device in dots per inch. The PySide2.QtGui.QPaintDevice.physicalDpiX() and PySide2.QtGui.QPaintDevice.physicalDpiY() functions also return the resolution of the device in dots per inch, but note that if the logical and physical resolution differ, the corresponding PySide2.QtGui.QPaintEngine must handle the mapping. Finally, the PySide2.QtGui.QPaintDevice.colorCount() function returns the number of different colors available for the paint device.

PySide2.QtGui.QPaintDevice.PaintDeviceMetric¶

Describes the various metrics of a paint device.

Constant Description
QPaintDevice.PdmWidth The width of the paint device in default coordinate system units (e.g. pixels for PySide2.QtGui.QPixmap and PySide2.QtWidgets.QWidget ). See also PySide2.QtGui.QPaintDevice.width() .
QPaintDevice.PdmHeight The height of the paint device in default coordinate system units (e.g. pixels for PySide2.QtGui.QPixmap and PySide2.QtWidgets.QWidget ). See also PySide2.QtGui.QPaintDevice.height() .
QPaintDevice.PdmWidthMM The width of the paint device in millimeters. See also PySide2.QtGui.QPaintDevice.widthMM() .
QPaintDevice.PdmHeightMM The height of the paint device in millimeters. See also PySide2.QtGui.QPaintDevice.heightMM() .
QPaintDevice.PdmNumColors The number of different colors available for the paint device. See also PySide2.QtGui.QPaintDevice.colorCount() .
QPaintDevice.PdmDepth The bit depth (number of bit planes) of the paint device. See also PySide2.QtGui.QPaintDevice.depth() .
QPaintDevice.PdmDpiX The horizontal resolution of the device in dots per inch. See also PySide2.QtGui.QPaintDevice.logicalDpiX() .
QPaintDevice.PdmDpiY The vertical resolution of the device in dots per inch. See also PySide2.QtGui.QPaintDevice.logicalDpiY() .
QPaintDevice.PdmPhysicalDpiX The horizontal resolution of the device in dots per inch. See also PySide2.QtGui.QPaintDevice.physicalDpiX() .
QPaintDevice.PdmPhysicalDpiY The vertical resolution of the device in dots per inch. See also PySide2.QtGui.QPaintDevice.physicalDpiY() .
QPaintDevice.PdmDevicePixelRatio The device pixel ratio for device. Common values are 1 for normal-dpi displays and 2 for high-dpi “retina” displays.
QPaintDevice.PdmDevicePixelRatioScaled The scaled device pixel ratio for the device. This is identical to , except that the value is scaled by a constant factor in order to support paint devices with fractional scale factors. The constant scaling factor used is devicePixelRatioFScale(). This enum value has been introduced in Qt 5.6.

PySide2.QtGui.QPaintDevice.painters¶
PySide2.QtGui.QPaintDevice.colorCount()¶
Return type:PySide2.QtCore.int

Returns the number of different colors available for the paint device. If the number of colors available is too great to be represented by the int data type, then INT_MAX will be returned instead.

PySide2.QtGui.QPaintDevice.depth()¶
Return type:PySide2.QtCore.int

Returns the bit depth (number of bit planes) of the paint device.

PySide2.QtGui.QPaintDevice.devType()¶
Return type:PySide2.QtCore.int

Returns the device type identifier, which is QInternal.Widget if the device is a PySide2.QtWidgets.QWidget , QInternal.Pixmap if it’s a PySide2.QtGui.QPixmap , QInternal.Printer if it’s a PySide2.QtPrintSupport.QPrinter , QInternal.Picture if it’s a PySide2.QtGui.QPicture , or QInternal.UnknownDevice in other cases.

PySide2.QtGui.QPaintDevice.devicePixelRatio()¶
Return type:PySide2.QtCore.int

Returns the device pixel ratio for device.

Common values are 1 for normal-dpi displays and 2 for high-dpi “retina” displays.

PySide2.QtGui.QPaintDevice.devicePixelRatioF()¶
Return type:PySide2.QtCore.qreal

Returns the device pixel ratio for the device as a floating point number.

static PySide2.QtGui.QPaintDevice.devicePixelRatioFScale()¶
Return type:PySide2.QtCore.qreal

Returns the scaling factor used for PdmDevicePixelRatioScaled . Classes that are not PySide2.QtGui.QPaintDevice subclasses are implementing PySide2.QtGui.QPaintDevice.metric() , and need to access this constant. Since it’s a qreal, it cannot be an enum, and an inline function is more efficient than a static member variable.

PySide2.QtGui.QPaintDevice.logicalDpiX()¶
Return type:PySide2.QtCore.int

Returns the horizontal resolution of the device in dots per inch, which is used when computing font sizes. For X11, this is usually the same as could be computed from PySide2.QtGui.QPaintDevice.widthMM() .

Note that if the doesn’t equal the PySide2.QtGui.QPaintDevice.physicalDpiX() , the corresponding PySide2.QtGui.QPaintEngine must handle the resolution mapping.

PySide2.QtGui.QPaintDevice.physicalDpiX()¶
Return type:PySide2.QtCore.int

Returns the horizontal resolution of the device in dots per inch. For example, when printing, this resolution refers to the physical printer’s resolution. The logical DPI on the other hand, refers to the resolution used by the actual paint engine.

Note that if the doesn’t equal the PySide2.QtGui.QPaintDevice.logicalDpiX() , the corresponding PySide2.QtGui.QPaintEngine must handle the resolution mapping.
