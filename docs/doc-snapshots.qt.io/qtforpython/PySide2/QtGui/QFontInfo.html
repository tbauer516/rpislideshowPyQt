Detailed Description¶

The PySide2.QtGui.QFontInfo class provides general information about fonts.

The PySide2.QtGui.QFontInfo class provides the same access functions as PySide2.QtGui.QFont , e.g. PySide2.QtGui.QFontInfo.family() , PySide2.QtGui.QFontInfo.pointSize() , PySide2.QtGui.QFontInfo.italic() , PySide2.QtGui.QFontInfo.weight() , PySide2.QtGui.QFontInfo.fixedPitch() , PySide2.QtGui.QFontInfo.styleHint() etc. But whilst the PySide2.QtGui.QFont access functions return the values that were set, a PySide2.QtGui.QFontInfo object returns the values that apply to the font that will actually be used to draw the text.

For example, when the program asks for a 25pt Courier font on a machine that has a non-scalable 24pt Courier font, PySide2.QtGui.QFont will (normally) use the 24pt Courier for rendering. In this case, QFont.pointSize() returns 25 and QFontInfo.pointSize() returns 24.

There are three ways to create a PySide2.QtGui.QFontInfo object.

  1. Calling the PySide2.QtGui.QFontInfo constructor with a PySide2.QtGui.QFont creates a font info object for a screen-compatible font, i.e. the font cannot be a printer font. If the font is changed later, the font info object is not updated.

    (Note: If you use a printer font the values returned may be inaccurate. Printer fonts are not always accessible so the nearest screen font is used if a printer font is supplied.)

  2. QWidget.fontInfo() returns the font info for a widget’s font. This is equivalent to calling PySide2.QtGui.QFontInfo (widget-> font() ). If the widget’s font is changed later, the font info object is not updated.

  3. QPainter.fontInfo() returns the font info for a painter’s current font. If the painter’s font is changed later, the font info object is not updated.

class PySide2.QtGui.QFontInfo(arg__1)¶
class PySide2.QtGui.QFontInfo(arg__1)
Parameters:arg__1 – PySide2.QtGui.QFont

Constructs a font info object for font .

The font must be screen-compatible, i.e. a font you use when drawing text in widgets or pixmaps , not PySide2.QtGui.QPicture or PySide2.QtPrintSupport.QPrinter .

The font info object holds the information for the font that is passed in the constructor at the time it is created, and is not updated if the font’s attributes are changed later.

Use QPainter.fontInfo() to get the font info when painting. This will give correct results also when painting on paint device that is not screen-compatible.

PySide2.QtGui.QFontInfo.rawMode()¶
Return type:PySide2.QtCore.bool

Returns true if the font is a raw mode font; otherwise returns false.

If it is a raw mode font, all other functions in PySide2.QtGui.QFontInfo will return the same values set in the PySide2.QtGui.QFont , regardless of the font actually used.
