Detailed Description¶

The PySide2.QtGui.QScrollPrepareEvent class is sent in preparation of scrolling.

The scroll prepare event is sent before scrolling (usually by PySide2.QtWidgets.QScroller ) is started. The object receiving this event should set PySide2.QtGui.QScrollPrepareEvent.viewportSize() , maxContentPos and PySide2.QtGui.QScrollPrepareEvent.contentPos() . It also should accept this event to indicate that scrolling should be started.

It is not guaranteed that a PySide2.QtGui.QScrollEvent will be sent after an acceepted PySide2.QtGui.QScrollPrepareEvent , e.g. in a case where the maximum content position is (0,0).
