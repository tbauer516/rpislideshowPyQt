Detailed Description¶

The PySide2.QtGui.QVector2D class represents a vector or vertex in 2D space.

The PySide2.QtGui.QVector2D class can also be used to represent vertices in 2D space. We therefore do not need to provide a separate vertex class.

Returns the distance that this vertex is from a line defined by point and the unit vector direction .

If direction is a null vector, then it does not define a line. In that case, the distance from point to this vertex is returned.

PySide2.QtGui.QVector2D.normalized()¶
Return type:PySide2.QtGui.QVector2D

Returns the normalized unit vector form of this vector.

If this vector is null, then a null vector is returned. If the length of the vector is very close to 1, then the vector will be returned as-is. Otherwise the normalized form of the vector of length 1 will be returned.
