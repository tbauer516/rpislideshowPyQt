Detailed Description¶

The PySide2.QtGui.QPolygonF class provides a vector of points using floating point precision.

A PySide2.QtGui.QPolygonF is a QVector < PySide2.QtCore.QPointF >. The easiest way to add points to a PySide2.QtGui.QPolygonF is to use its streaming operator, as illustrated below:

polygon = QPolygonF()
polygon << QPointF(10.4, 20.5) << QPointF(20.2, 30.2)

In addition to the functions provided by QVector , PySide2.QtGui.QPolygonF provides the PySide2.QtGui.QPolygonF.boundingRect() and PySide2.QtGui.QPolygonF.translate() functions for geometry operations. Use the QMatrix.map() function for more general transformations of QPolygonFs.

PySide2.QtGui.QPolygonF also provides the PySide2.QtGui.QPolygonF.isClosed() function to determine whether a polygon’s start and end points are the same, and the PySide2.QtGui.QPolygonF.toPolygon() function returning an integer precision copy of this polygon.

The PySide2.QtGui.QPolygonF class is implicitly shared .
