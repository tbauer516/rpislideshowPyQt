Detailed Description¶

The PySide2.QtGui.QVector4D class represents a vector or vertex in 4D space.

The PySide2.QtGui.QVector4D class can also be used to represent vertices in 4D space. We therefore do not need to provide a separate vertex class.

Normalizes the currect vector in place. Nothing happens if this vector is a null vector or the length of the vector is very close to 1.

PySide2.QtGui.QVector4D.normalized()¶
Return type:PySide2.QtGui.QVector4D

Returns the normalized unit vector form of this vector.

If this vector is null, then a null vector is returned. If the length of the vector is very close to 1, then the vector will be returned as-is. Otherwise the normalized form of the vector of length 1 will be returned.
