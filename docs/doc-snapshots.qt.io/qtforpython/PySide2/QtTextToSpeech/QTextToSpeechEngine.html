Detailed Description¶

The PySide2.QtTextToSpeech.QTextToSpeechEngine class is the base for text-to-speech engine integrations.

An engine implementation must derive from PySide2.QtTextToSpeech.QTextToSpeechEngine and implement all its pure virtual methods.
