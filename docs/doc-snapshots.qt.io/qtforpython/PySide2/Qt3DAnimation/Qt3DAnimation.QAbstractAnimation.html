Detailed Description¶

An abstract base class for Qt3D animations

Qt3DAnimation.QAbstractAnimation is an abstract base class for all animations. Qt3DAnimation.QAbstractAnimation can not be directly instantiated, but rather through its subclasses. PySide2.QtCore.QAbstractAnimation specifies common properties for all Qt3D animations, such as animation name and type, current position and animation duration, while leaving the actual animating for the subclasses.
