QTcpSocket¶

Inherited by: QSslSocket

Detailed Description¶

The PySide2.QtNetwork.QTcpSocket class provides a TCP socket.

TCP (Transmission Control Protocol) is a reliable, stream-oriented, connection-oriented transport protocol. It is especially well suited for continuous transmission of data.

PySide2.QtNetwork.QTcpSocket is a convenience subclass of PySide2.QtNetwork.QAbstractSocket that allows you to establish a TCP connection and transfer streams of data. See the PySide2.QtNetwork.QAbstractSocket documentation for details.

Note

TCP sockets cannot be opened in QIODevice.Unbuffered mode.
