Detailed Description¶

The PySide2.QtNetwork.QDnsDomainNameRecord class stores information about a domain name record.

When performing a name server lookup, zero or more records will be returned. Each record is represented by a PySide2.QtNetwork.QDnsDomainNameRecord instance.
