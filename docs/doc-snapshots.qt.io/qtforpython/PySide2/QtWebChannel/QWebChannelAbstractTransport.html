QWebChannelAbstractTransport¶

Detailed Description¶

Communication channel between the C++ PySide2.QtWebChannel.QWebChannel server and a HTML/JS client.

Users of the PySide2.QtWebChannel.QWebChannel must implement this interface and connect instances of it to the PySide2.QtWebChannel.QWebChannel server for every client that should be connected to the PySide2.QtWebChannel.QWebChannel . The Qt WebChannel Standalone Example shows how this can be done using Qt WebSockets .

Note

The JSON message protocol is considered internal and might change over time.
