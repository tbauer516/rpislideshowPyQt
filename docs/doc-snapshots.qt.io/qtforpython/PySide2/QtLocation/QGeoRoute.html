Detailed Description¶

The PySide2.QtLocation.QGeoRoute class represents a route between two points.

A PySide2.QtLocation.QGeoRoute object contains high level information about a route, such as the length the route, the estimated travel time for the route, and enough information to render a basic image of the route on a map.

The PySide2.QtLocation.QGeoRoute object also contains a list of PySide2.QtLocation.QGeoRouteSegment objecs which describe subsections of the route in greater detail.

Routing information is normally requested using QGeoRoutingManager.calculateRoute() , which returns a PySide2.QtLocation.QGeoRouteReply instance. If the operation is completed successfully the routing information can be accessed with QGeoRouteReply.routes()
