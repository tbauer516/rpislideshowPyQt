Detailed Description¶

The PySide2.QtLocation.QPlaceImage class represents a reference to an image.

Each PySide2.QtLocation.QPlaceImage represents a set of metadata about an image such as it’s url, identifier and MIME type. These are properties in addition to those provided by PySide2.QtLocation.QPlaceContent .

Note: The Places API only supports images as ‘retrieve-only’ objects. Submitting images to a provider is not a supported use case.
