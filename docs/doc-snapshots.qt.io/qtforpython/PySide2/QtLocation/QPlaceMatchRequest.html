Detailed Description¶

The PySide2.QtLocation.QPlaceMatchRequest class is used to find places from one manager that match those from another. It represents a set of request parameters.

Places from another manager that may have corresponding/matching places in the current manager are assigned using PySide2.QtLocation.QPlaceMatchRequest.setPlaces() or PySide2.QtLocation.QPlaceMatchRequest.setResults() . A set of further parameters are specified which determines the criteria for matching.

The typical key for matching is the QPlaceMatchRequest::AlternativeId , the value is an alternative identifier attribute type of the format x_id_<provider name> for example x_id_here. The provider name is name supplied to the PySide2.QtLocation.QGeoServiceProvider instance.

See Matching places between managers for an example on how to use a match request.
