Detailed Description¶

The PySide2.QtLocation.QPlaceReview class represents a review of a place.

Each PySide2.QtLocation.QPlaceReview has a number of properties such as a title, text, date of submission and rating; in addition to those properties inherited from PySide2.QtLocation.QPlaceContent .

Note: The Places API only supports reviews as ‘retrieve-only’ objects. Submitting reviews to a provider is not a supported use case.
