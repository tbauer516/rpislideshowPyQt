QHolsterFilter¶

Detailed Description¶

The PySide2.QtSensors.QHolsterFilter class is a convenience wrapper around PySide2.QtSensors.QSensorFilter .

The only difference is that the PySide2.QtSensors.QHolsterFilter.filter() method features a pointer to PySide2.QtSensors.QHolsterReading instead of PySide2.QtSensors.QSensorReading .

class PySide2.QtSensors.QHolsterFilter¶
PySide2.QtSensors.QHolsterFilter.filter(reading)¶
Parameters:reading – PySide2.QtSensors.QHolsterReading
Return type:PySide2.QtCore.bool

Called when reading changes. Returns false to prevent the reading from propagating.
