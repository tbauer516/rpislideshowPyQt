Detailed Description¶

The PySide2.QtSensors.QTapSensor class is a convenience wrapper around PySide2.QtSensors.QSensor .

The only behavioural difference is that this class sets the type properly.

This class also features a PySide2.QtSensors.QTapSensor.reading() function that returns a PySide2.QtSensors.QTapReading instead of a PySide2.QtSensors.QSensorReading .

For details about how the sensor works, see PySide2.QtSensors.QTapReading .
