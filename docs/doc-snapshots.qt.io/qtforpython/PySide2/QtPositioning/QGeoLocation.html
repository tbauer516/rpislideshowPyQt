Detailed Description¶

The PySide2.QtPositioning.QGeoLocation class represents basic information about a location.

A PySide2.QtPositioning.QGeoLocation consists of a coordinate and corresponding address, along with an optional bounding box which is the recommended region to be displayed when viewing the location.

PySide2.QtPositioning.QGeoLocation.boundingBox()¶
Return type:PySide2.QtPositioning.QGeoRectangle

Returns a bounding box which represents the recommended region to display when viewing this location.

For example, a building’s location may have a region centered around the building, but the region is large enough to show it’s immediate surrounding geographical context.
