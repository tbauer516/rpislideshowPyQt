Detailed Description¶

The PySide2.QtPositioning.QGeoCircle class defines a circular geographic area.

The circle is defined in terms of a PySide2.QtPositioning.QGeoCoordinate which specifies the center of the circle and a qreal which specifies the radius of the circle in meters.

The circle is considered invalid if the center coordinate is invalid or if the radius is less than zero.

This class is a Q_GADGET() since Qt 5.5. It can be directly used from C++ and QML .
