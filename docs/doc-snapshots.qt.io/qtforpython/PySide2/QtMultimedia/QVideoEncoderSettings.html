Detailed Description¶

The PySide2.QtMultimedia.QVideoEncoderSettings class provides a set of video encoder settings.

A video encoder settings object is used to specify the video encoder settings used by PySide2.QtMultimedia.QMediaRecorder . Video encoder settings are selected by constructing a PySide2.QtMultimedia.QVideoEncoderSettings object, setting the desired properties and then passing it to a PySide2.QtMultimedia.QMediaRecorder instance using the QMediaRecorder.setEncodingSettings() function.

QVideoEncoderSettings videoSettings;
videoSettings.setCodec("video/mpeg2");
videoSettings.setResolution(640, 480);

recorder->setVideoSettings(videoSettings);
