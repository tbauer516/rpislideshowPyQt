QCameraCaptureDestinationControl¶

Detailed Description¶

The PySide2.QtMultimedia.QCameraCaptureDestinationControl class provides a control for setting capture destination.

Depending on backend capabilities capture to file, buffer or both can be supported.

The interface name of PySide2.QtMultimedia.QCameraCaptureDestinationControl is org.qt-project.qt.cameracapturedestinationcontrol/5.0 as defined in QCameraCaptureDestinationControl_iid() .
