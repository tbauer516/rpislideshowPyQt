QAbstractAudioInput¶

Detailed Description¶

The PySide2.QtMultimedia.QAbstractAudioInput class provides access for PySide2.QtMultimedia.QAudioInput to access the audio device provided by the plugin.

QAudioDeviceInput keeps an instance of PySide2.QtMultimedia.QAbstractAudioInput and routes calls to functions of the same name to PySide2.QtMultimedia.QAbstractAudioInput . This means that it is PySide2.QtMultimedia.QAbstractAudioInput that implements the audio functionality. For a description of the functionality, see the PySide2.QtMultimedia.QAudioInput class description.
