QAbstractAudioOutput¶

Detailed Description¶

The PySide2.QtMultimedia.QAbstractAudioOutput class is a base class for audio backends.

PySide2.QtMultimedia.QAbstractAudioOutput implements audio functionality for PySide2.QtMultimedia.QAudioOutput , i.e., PySide2.QtMultimedia.QAudioOutput routes function calls to PySide2.QtMultimedia.QAbstractAudioOutput . For a description of the functionality that is implemented, see the PySide2.QtMultimedia.QAudioOutput class and function descriptions.
