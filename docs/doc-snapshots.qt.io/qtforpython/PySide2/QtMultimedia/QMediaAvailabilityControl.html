QMediaAvailabilityControl¶

Detailed Description¶

The PySide2.QtMultimedia.QMediaAvailabilityControl class supplies a control for reporting availability of a service.

An instance of PySide2.QtMultimedia.QMediaObject (or its derived classes) can report any changes in availability via this control.

The interface name of PySide2.QtMultimedia.QMediaAvailabilityControl is org.qt-project.qt.mediaavailabilitycontrol/5.0 as defined in QMediaAvailabilityControl_iid() .

PySide2.QtMultimedia.QMediaAvailabilityControl.availability()¶
Return type:PySide2.QtMultimedia.QMultimedia.AvailabilityStatus

Returns the current availability of this instance of the media service. If the availability changes at run time (for example, some other media client takes all media resources) the availabilityChanges() signal should be emitted.
