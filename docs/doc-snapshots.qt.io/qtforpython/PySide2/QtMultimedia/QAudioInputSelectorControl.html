QAudioInputSelectorControl¶

Detailed Description¶

The PySide2.QtMultimedia.QAudioInputSelectorControl class provides an audio input selector media control.

The PySide2.QtMultimedia.QAudioInputSelectorControl class provides descriptions of the audio inputs available on a system and allows one to be selected as the audio input of a media service.

The interface name of PySide2.QtMultimedia.QAudioInputSelectorControl is org.qt-project.qt.audioinputselectorcontrol/5.0 as defined in QAudioInputSelectorControl_iid() .
