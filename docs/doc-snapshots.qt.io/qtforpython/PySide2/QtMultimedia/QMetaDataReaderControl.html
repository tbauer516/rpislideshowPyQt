QMetaDataReaderControl¶

Detailed Description¶

The PySide2.QtMultimedia.QMetaDataReaderControl class provides read access to the meta-data of a PySide2.QtMultimedia.QMediaService ‘s media.

If a PySide2.QtMultimedia.QMediaService can provide read or write access to the meta-data of its current media it will implement PySide2.QtMultimedia.QMetaDataReaderControl . This control provides functions for both retrieving and setting meta-data values. Meta-data may be addressed by the keys defined in the QMediaMetaData namespace.

The functionality provided by this control is exposed to application code by the meta-data members of PySide2.QtMultimedia.QMediaObject , and so meta-data access is potentially available in any of the media object classes. Any media service may implement PySide2.QtMultimedia.QMetaDataReaderControl .

The interface name of PySide2.QtMultimedia.QMetaDataReaderControl is org.qt-project.qt.metadatareadercontrol/5.0 as defined in QMetaDataReaderControl_iid() .
