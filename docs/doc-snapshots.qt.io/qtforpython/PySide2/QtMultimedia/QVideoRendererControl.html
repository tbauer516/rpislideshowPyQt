QVideoRendererControl¶

Detailed Description¶

The PySide2.QtMultimedia.QVideoRendererControl class provides a media control for rendering video to a PySide2.QtMultimedia.QAbstractVideoSurface .

Using the PySide2.QtMultimedia.QVideoRendererControl.surface() property of PySide2.QtMultimedia.QVideoRendererControl a PySide2.QtMultimedia.QAbstractVideoSurface may be set as the video render target of a PySide2.QtMultimedia.QMediaService .

QVideoRendererControl *rendererControl = mediaService->requestControl<QVideoRendererControl *>();
rendererControl->setSurface(myVideoSurface);

PySide2.QtMultimedia.QVideoRendererControl is one of a number of possible video output controls.

The interface name of PySide2.QtMultimedia.QVideoRendererControl is org.qt-project.qt.videorenderercontrol/5.0 as defined in QVideoRendererControl_iid() .
