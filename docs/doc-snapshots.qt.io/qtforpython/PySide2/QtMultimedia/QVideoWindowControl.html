QVideoWindowControl¶

Detailed Description¶

The PySide2.QtMultimedia.QVideoWindowControl class provides a media control for rendering video to a window.

The PySide2.QtMultimedia.QVideoWindowControl.winId() property PySide2.QtMultimedia.QVideoWindowControl allows a platform specific window ID to be set as the video render target of a PySide2.QtMultimedia.QMediaService . The PySide2.QtMultimedia.QVideoWindowControl.displayRect() property is used to set the region of the window the video should be rendered to, and the PySide2.QtMultimedia.QVideoWindowControl.aspectRatioMode() property indicates how the video should be scaled to fit the PySide2.QtMultimedia.QVideoWindowControl.displayRect() .

QVideoWindowControl *windowControl = mediaService->requestControl<QVideoWindowControl *>();
windowControl->setWinId(widget->winId());
windowControl->setDisplayRect(widget->rect());
windowControl->setAspectRatioMode(Qt::KeepAspectRatio);

PySide2.QtMultimedia.QVideoWindowControl is one of a number of possible video output controls.

The interface name of PySide2.QtMultimedia.QVideoWindowControl is org.qt-project.qt.videowindowcontrol/5.0 as defined in QVideoWindowControl_iid() .
