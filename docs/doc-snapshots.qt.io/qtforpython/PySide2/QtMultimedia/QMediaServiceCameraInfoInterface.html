QMediaServiceCameraInfoInterface¶

Detailed Description¶

The PySide2.QtMultimedia.QMediaServiceCameraInfoInterface class interface provides camera-specific information about devices supported by a camera service plug-in.

A QMediaServiceProviderPlugin may implement this interface, in that case it also needs to implement the PySide2.QtMultimedia.QMediaServiceSupportedDevicesInterface .
