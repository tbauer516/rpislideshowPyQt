QMediaServiceDefaultDeviceInterface¶

Detailed Description¶

The PySide2.QtMultimedia.QMediaServiceDefaultDeviceInterface class interface identifies the default device used by a media service plug-in.

A QMediaServiceProviderPlugin may implement this interface.

class PySide2.QtMultimedia.QMediaServiceDefaultDeviceInterface¶
PySide2.QtMultimedia.QMediaServiceDefaultDeviceInterface.defaultDevice(service)¶
Parameters:service – PySide2.QtCore.QByteArray
Return type:PySide2.QtCore.QByteArray

Returns the default device for a service type.
