QAudioOutputSelectorControl¶

Detailed Description¶

The PySide2.QtMultimedia.QAudioOutputSelectorControl class provides an audio output selector media control.

The PySide2.QtMultimedia.QAudioOutputSelectorControl class provides descriptions of the audio outputs available on a system and allows one to be selected as the audio output of a media service.

The interface name of PySide2.QtMultimedia.QAudioOutputSelectorControl is org.qt-project.qt.audiooutputselectorcontrol/5.0 as defined in QAudioOutputSelectorControl_iid() .
