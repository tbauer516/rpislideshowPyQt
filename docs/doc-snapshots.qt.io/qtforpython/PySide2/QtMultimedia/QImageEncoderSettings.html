Detailed Description¶

The PySide2.QtMultimedia.QImageEncoderSettings class provides a set of image encoder settings.

A image encoder settings object is used to specify the image encoder settings used by PySide2.QtMultimedia.QCameraImageCapture . Image encoder settings are selected by constructing a PySide2.QtMultimedia.QImageEncoderSettings object, setting the desired properties and then passing it to a PySide2.QtMultimedia.QCameraImageCapture instance using the QCameraImageCapture::setImageSettings() function.

QImageEncoderSettings imageSettings;
imageSettings.setCodec("image/jpeg");
imageSettings.setResolution(1600, 1200);

imageCapture->setEncodingSettings(imageSettings);
