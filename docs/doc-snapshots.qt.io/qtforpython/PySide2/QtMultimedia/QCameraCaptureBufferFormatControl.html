QCameraCaptureBufferFormatControl¶

Detailed Description¶

The PySide2.QtMultimedia.QCameraCaptureBufferFormatControl class provides a control for setting the capture buffer format.

The format is of type QVideoFrame.PixelFormat .

The interface name of PySide2.QtMultimedia.QCameraCaptureBufferFormatControl is org.qt-project.qt.cameracapturebufferformatcontrol/5.0 as defined in QCameraCaptureBufferFormatControl_iid() .
