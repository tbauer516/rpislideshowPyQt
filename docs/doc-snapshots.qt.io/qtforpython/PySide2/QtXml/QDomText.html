Detailed Description¶

The PySide2.QtXml.QDomText class represents text data in the parsed XML document.

You can split the text in a PySide2.QtXml.QDomText object over two PySide2.QtXml.QDomText objecs with PySide2.QtXml.QDomText.splitText() .

For further information about the Document Object Model see Level 1 and Level 2 Core. For a more general introduction of the DOM implementation see the PySide2.QtXml.QDomDocument documentation.

class PySide2.QtXml.QDomText¶
class PySide2.QtXml.QDomText(x)
Parameters:x – PySide2.QtXml.QDomText

Constructs an empty PySide2.QtXml.QDomText object.

To construct a PySide2.QtXml.QDomText with content, use QDomDocument.createTextNode() .

Constructs a copy of x .

The data of the copy is shared (shallow copy): modifying one node will also change the other. If you want to make a deep copy, use PySide2.QtXml.QDomNode.cloneNode() .

PySide2.QtXml.QDomText.splitText(offset)¶
Parameters:offset – PySide2.QtCore.int
Return type:PySide2.QtXml.QDomText

Splits this DOM text object into two PySide2.QtXml.QDomText objects. This object keeps its first offset characters and the second (newly created) object is inserted into the document tree after this object with the remaining characters.

The function returns the newly created object.
