The Qt XML module provides C++ implementations of the SAX and DOM standards for XML.

The module is not actively maintained anymore. Please use the PySide2.QtCore.QXmlStreamReader and PySide2.QtCore.QXmlStreamWriter classes in Qt Core instead.
