Detailed Description¶

The PySide2.QtMultimedia.QCamera class defines a view point through which the scene will be rendered.
class PySide2.Qt3DRender.Qt3DRender.QCamera([parent=nullptr])¶
Parameters:parent – PySide2.Qt3DCore.Qt3DCore::QNode

Creates a new PySide2.QtMultimedia.QCamera instance with the specified parent .

PySide2.Qt3DRender.Qt3DRender.QCamera.CameraTranslationOption¶

This enum specifies how camera view center is translated

Constant Description
Qt3DRender.QCamera.TranslateViewCenter Translate the view center causing the view direction to remain the same
Qt3DRender.QCamera.DontTranslateViewCenter Don’t translate the view center causing the view direction to change
